# SVGEditor - SVG编辑器

![SVGEditor](https://img.shields.io/badge/SVGEditor-v1.0-orange)
![License](https://img.shields.io/badge/license-MIT-green)

一个充满活力的橙色创意风格 SVG 在线编辑器，支持实时预览、多种形状插入和模板功能。

## 功能特点

- 实时预览 - 编辑代码的同时即时查看效果
- 多种形状 - 矩形、圆形、椭圆、直线、多边形、路径、文字
- 精美模板 - 星形、心形、箭头、云朵、闪电、月亮
- 多语言支持 - 中文、英语、西班牙语、日语、韩语
- 网格视图 - 可切换的设计辅助网格
- 代码格式化 - 自动美化 SVG 代码
- 撤销功能 - 支持撤销操作
- 下载导出 - 一键下载 SVG 文件

## 在线使用

https://kanshaowz.github.io/svgeditor

## 本地使用

```bash
# 克隆仓库
git clone https://github.com/kanshaowz/svgeditor.git

# 进入目录
cd svgeditor

# 打开 index.html 即可使用
```
